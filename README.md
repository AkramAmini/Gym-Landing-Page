# Gym Landing Page

A modern, fully responsive landing page for a fitness club. This project focuses on utilizing **Bootstrap 5** for rapid layout development and custom CSS for sleek typography and animations.

## Features
- **Responsive Navigation:** Smooth mobile-friendly hamburger menu built with Bootstrap Navbar.
- **Dynamic Grid Layout:** Pricing plans, trainer profiles, and features structured perfectly using Bootstrap's grid system (`.row`, `.col-*`).
- **Interactive Components:** Custom hover effects on pricing cards and buttons.
- **Clean Semantic HTML:** High-quality markup for better SEO and accessibility.

##  Tech Stack
- **HTML5** (Semantic elements)
- **CSS3** (Custom properties, flexbox, transitions)
- **Bootstrap 5** (Grid, Navbar, Cards, Utility classes)
